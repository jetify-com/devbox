# devbox add

Add a new package to your devbox

```bash
devbox add <pkg>... [flags]
```

## Options

<!-- Markdown Table of Options -->
| Option | Description |
| --- | --- |
| `-c, --config string` | path to directory containing a devbox.json config file |
| `-h, --help` | help for add |
| `-q, --quiet` | Quiet mode: Suppresses logs. |
