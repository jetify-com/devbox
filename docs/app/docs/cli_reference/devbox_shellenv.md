# devbox shellenv

Print shell commands that add Devbox packages to your PATH

```bash
devbox shellenv [flags]
```

## Options

```text
  -c, --config string   path to directory containing a devbox.json config file
  -h, --help            help for shellen  
  -q, --quiet   suppresses logs``
```
